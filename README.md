# Parser Deubaso-Composifity

## About the project
A parser of videos from porn sites. Downloads videos only in the best quality. Beginning of development 06.01.2026.

## Getting Started
You can use the parser right away.

### Create
If you want a dictionary with video links, create a `data.json` file and write the following code, then add your links to the dictionary:
```json
{
  "videos":
    {
      "1": "https://vps402.strip2.co/video/...",
      "2": "https://rt.xgroovy.com/videos/...",
      "3": "https://anal.media/ru/video/..."
    }
}
```
